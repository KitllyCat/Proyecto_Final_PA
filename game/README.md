# Remoria

Remoria es una **novela visual narrativa (visual novel)** desarrollada por Ko-Lab:Mode, centrada en la memoria, las decisiones cotidianas y la forma en que pequeños actos pueden alterar el rumbo de una historia. A través de una estética tranquila y una narración introspectiva, el jugador acompaña a **Kami**, un niño que revive una etapa de su vida con una perspectiva que no debería tener todavía.

La historia se construye mediante escenas interactivas, diálogos y elecciones que influyen en el desarrollo emocional del protagonista. Remoria no busca castigar ni premiar de forma tradicional, sino invitar a la reflexión.

## Estado del proyecto

El proyecto **ya cuenta con un ejecutable funcional**, listo para ser usado.

Actualmente se encuentra en la version Beta v0.6.9+

No es necesario compilar ni configurar dependencias adicionales para jugar. Simplemente ejecuta el archivo correspondiente y comienza la experiencia...

## Estructura general

Dentro de esta carpeta (`game/`) encontrarás:

- El **ejecutable del juego**
- La carpeta `data/` con escenas, diálogos y configuración narrativa
- La carpeta `assets/` con imágenes, audio y recursos gráficos

No modifique ninguno de sus archivos, pueden ocurrir errores en el ejecutable.

## Nota del autor

Remoria es una historia pensada para avanzar con calma. Algunas decisiones no se anuncian como tales, pero todas dejan una huella. Observa, escucha y tómate tu tiempo...
